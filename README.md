# obj-viewer (WIP)
An OBJ file viewer that uses software rasterization.

## Usage
Drag an OBJ file to the app window. The program will open the file.

### Keys
- W: Move forward
- A: Strafe left
- S: Move backward
- D: Strafe right
- Left: Look left
- Right: Look right
- Up: Look up
- Down: Look down
- F2: Take a screenshot

### Remarks
- This program only supports files with only triangles as faces
